# MinePaperEngine

![Discord](https://img.shields.io/discord/1370807259495534663?style=flat-square&logo=discord&label=xkball's%20mods&link=https%3A%2F%2Fdiscord.gg%2FS9DBXWHNsc) ![GitHub License](https://img.shields.io/github/license/xkball/MinePaperEngine?style=flat-square)

Turn your Minecraft window into a desktop wallpaper.

After launching the game, a tray icon will appear. Click it to toggle between running the game as a normal window or as a dynamic wallpaper.

The wallpaper mode does not pause the game or allow gameplay. Please manually limit the frame rate or hide the HUD before entering wallpaper mode.

Currently only works on Windows systems. On other systems, using this mod should theoretically have no effect.
